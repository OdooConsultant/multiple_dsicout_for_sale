Multi Discount Sale

3 Levels Discount on Sales

Description

Shows a 2 new level of sale discount reflected in accounting

This module shows a 2 other level of discount in the sale order.

Discounts are created and edited directly on the quotation and sale order lines.

Both discounts are printed in PDF Quote, and both are displayed in PDF if they are covered in form, otherwise not.

Installation

To install this module, you need to:
  • You must have installed or available to install the modules sale and account.

Usage

To use this module, you need to:
  • Check that your sales configuration in Odoo has checked Discounts.
